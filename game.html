INSTRUCTIONS

Welcome to "Guess The Word"! Please select a difficulty where you'll presented a certain amount of lives and the objective of this game is to select one letter at time in order to try to correctly guess what the random word is. If you select an incorrect letter, a life will be lost. However if you select a correct letterthe letter will appear in the world at it's correct location. If you lose all your lives before you guess the word then you lose. Also, each game you'll be given one free hint if you choose to use it. There's also a second hint that's avaliable, however a life will be lost if used. GOOD LUCK AND HAVE FUN!
